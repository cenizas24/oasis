# oasis  

Com navegar per la web Oasis:  
Primera Pantalla  (Home.vue)  

En obrir la web veuràs el logo gran i si fas scroll cap baix trobràs el nostre vinil giratori, que té un botó al mig on posa "PLAY", quan apretes apareix una animació d'explosió de notes musicals i sona una pista d'Oasis editada perquè sembli que està trencada, fent referència a les pistes que el grup posa en mig dels seus albums. Pots parar la pista d'audio apretant al botó de "STOP".  
Menú vertical (esquerra): cada botó porta a una secció  
ArTisTEs → /artistes  
vInIls → /vinils  
GèNeRes → /generes  
OfErtEs → /ofertes  
AboUt → /about  
  
Pàgina Vinils (Vinils.vue)
Graella de vinils que mostra la llista feta amb json, dels 8 vinils, si cliques a qualsevol d'aquells et surt una targeta amb tota la informació de cada vinil, has d'entrar dues vegades a la targeta per revelar el Album.  

Pàgina Gèneres (Gèneres.vue)
4 fotografies grans, que reprersenten 4 estils musicals, si apretes a qualsevol d'ells sona una pista d'audio trencada representant cada gènere musical.  

Pàgina Artistes (Artistes.vue)
Títol gran de la secció, i una breu explicació d'aquesta secció.  
Quatre imatges d'una icona, si pases amb el cursor per sobre es revelen els Artistes d'aquest mes.  

Pàgina Ofertes (Ofertes.vue)  
Un text encriptat per una partitura musical, si apretes al botó de "Revela el missatge" es revela el text, i t'informa que tens un 15% de descompte als Vinils/Albums dels artistes mensuals.  

Pàgina About (About.vue)
Text de descripció de la nostra pàgina web i 3 vinils clicables que obren petites notes amb 3 motivacions per fer la web.  

El control de tornar que hi ha a totes les pàgines del menú, és un botó per poder retornar a la pàgina principal.
